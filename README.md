# Propose-Website-
Propose is a romantic, interactive website created to ask one special question in a memorable way. The  subtle micro-interactions, and soft motion. It culminates in a reveal screen—“Will you marry me?”—complete with animated confetti, ambient music (tap-to-play), and a choice of “Yes/Absolutely!” buttons.
